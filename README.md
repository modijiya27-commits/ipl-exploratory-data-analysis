# 🏏 IPL Data Analysis

This project explores historical Indian Premier League (IPL) data using Exploratory Data Analysis (EDA).

The analysis focuses on identifying patterns related to team performance, player achievements, toss decisions, match outcomes, venue trends, and scoring behavior using real IPL datasets.

## Key Questions

- Which teams have won the most matches?
- Which players have won the most Player of the Match awards?
- Does winning the toss influence match results?
- Which cities host the most IPL matches?
- How are match-winning margins distributed?
- What relationships exist between numerical match variables?
